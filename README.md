# Веб-студия Хакиева Магомета | khakiev.ru

Личный сайт-портфолио и лендинг студии веб-разработки. Проект представляет собой высокопроизводительный одностраничный сайт, написанный на чистом коде без использования тяжелых UI-библиотек или фреймворков. 

## 🌐 О проекте
Сайт решает сразу несколько бизнес-задач:
* **Презентация услуг:** Демонстрация опыта (8 лет, 120+ проектов, команда из 6 человек) и основных направлений работы (от лендингов до веб-приложений).
* **Конверсия:** Встроенная контактная форма и генерация vCard для быстрой связи.
* **Прозрачность:** Интерактивный калькулятор для предварительной оценки стоимости проекта.

## 🚀 Ключевые фичи и функционал
* **3D Canvas Анимация:** На главном экране реализована интерактивная 3D-сфера из частиц, которая реагирует на движение мыши.
* **Собственная Админ-панель:** Скрытая панель управления (доступ по паролю), позволяющая добавлять и удалять медиа-материалы. Данные реактивно обновляются и сохраняются в `localStorage`.
* **Интерактивный калькулятор:** Написан на Vanilla JS, динамически пересчитывает смету и сроки в зависимости от выбранного типа сайта и дополнительных услуг.
* **Динамический рендеринг:** Карточки портфолио, кейсы и отзывы хранятся в виде массивов данных (JSON) и генерируются через JavaScript, что делает код чище и упрощает масштабирование.
* **Кастомная UI-система:** 
  * Глубоко проработанная тёмная тема (Dark UI) с использованием CSS-переменных.
  * 3D Tilt-эффекты для карточек услуг и портфолио (меняют угол наклона в зависимости от положения курсора).
  * Анимации появления элементов при скролле (на базе `IntersectionObserver`).
  * Кастомный курсор с эффектом свечения (mix-blend-mode).
* **Интеграция заявок:** Контактная форма работает без бэкенда через AJAX-запросы FormSubmit, отправляя данные напрямую на email.

## 🛠 Стек технологий
Проект намеренно реализован в одном файле `index.html` (включая стили и скрипты) для обеспечения мгновенной загрузки.

* **Разметка:** HTML5 (семантические теги, доступность).
* **Стили:** CSS3 (CSS Grid, Flexbox, Backdrop-filter, кастомный скроллбар).
* **Логика:** Vanilla JavaScript (ES6+).
* **Типографика:** Google Fonts (Unbounded, Manrope, JetBrains Mono).
